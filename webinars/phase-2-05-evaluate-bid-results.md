# Webinar: Evaluate Bid Results

**Phase:** Phase 2 - Bidding (Repeat)  
**Duration:** 30-45 minutes

## Learning Objectives

- Understand where to find award announcements
- Interpret award results
- Request and conduct debriefings effectively

**To Understand Evaluation:**

**Requesting Debriefings:**

**Written Request Format:**
- Formal written request
- Email or letter
- Professional tone
- Include solicitation number
- Request specific information
- Set meeting time

**Who to Contact:**
- Contracting officer
- Contract specialist
- Point of contact from solicitation
- Small business specialist
- Agency contact

**Timing Requirements:**
- Request within timeframe (usually 3-5 days)
- Allow time for scheduling
- Agency will schedule
- Be flexible with timing
- Follow up if no response

**What to Request:**
- Evaluation summary
- Strengths and weaknesses
- How winner was selected
- Competitive position
- Improvement suggestions
- Clarifications
