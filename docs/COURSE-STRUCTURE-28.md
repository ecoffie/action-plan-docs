## Resource Path Convention

Resource names in the mapping (e.g. `consultant-phone-script.docx`) correspond to files in the `handouts/` folder. When the website serves them, paths are `/handouts/{filename}`. Source files (Vault, assets) are mapped in `handouts/README.md`.
