# Design System & Brand Guidelines

## Overview
This document establishes the visual design system for all Federal Contracting Roadmap Webinar Series assets.

## Brand Colors

### Primary Palette
- **Primary Dark Green**: #1a4d2e (Dark Forest Green - Growth, Stability, Trust)
- **Secondary Green**: #2d5016 (Medium Green - Confidence, Success)
- **Accent Gold**: #d4af37 (Gold - Excellence, Achievement, Premium)
- **Accent Light Green**: #90c695 (Light Green - Freshness, Growth)

### Neutral Palette
- **Dark Gray**: #333333 (Text - Primary)
- **Medium Gray**: #666666 (Text - Secondary)
- **Light Gray**: #f5f5f5 (Backgrounds)
- **White**: #ffffff (Backgrounds, Contrast)

### Status Colors
- **Success**: #2d5016 (Medium Green)
- **Warning**: #d4af37 (Gold)
- **Error**: #c92a2a (Red)
- **Info**: #1a4d2e (Dark Green)

## Typography

### Headings
- **Primary Font**: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, 'Helvetica Neue', Arial, sans-serif
- **H1**: 32-36pt, Bold, Primary Dark Green (#1a4d2e)
- **H2**: 24-28pt, Bold, Primary Dark Green (#1a4d2e)
- **H3**: 20-22pt, Semi-Bold, Dark Gray
- **H4**: 16-18pt, Semi-Bold, Dark Gray

### Body Text
- **Font**: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif
- **Size**: 11-12pt (documents), 14-16pt (web/presentations)
- **Line Height**: 1.6
- **Color**: Dark Gray (#333333)

### Special Text
- **Emphasis**: Bold, Primary Dark Green (#1a4d2e)
- **Links**: Primary Dark Green (#1a4d2e), Underlined
- **Captions**: 10pt, Medium Gray

## Layout Guidelines

### Document Margins
- **Standard Documents**: 1 inch all sides
- **Presentations**: Full bleed with safe zones
- **Checklists**: 0.75 inch margins for compact format

### Spacing
- **Section Spacing**: 24-32pt between major sections
- **Paragraph Spacing**: 12-16pt
- **List Item Spacing**: 8pt
- **Element Padding**: 16-20pt

### Grid System
- **2-column layout**: For comparison content
- **Single column**: For narrative content
- **3-column**: For feature lists, benefits

## Visual Elements

### Icons & Graphics
- **Style**: Minimalist, professional
- **Usage**: Consistent icon library
- **Color**: Primary Blue or Medium Gray
- **Size**: 16-24px for inline, 48-64px for featured

### Header/Footer Design
- **Header**: Brand name/logo, document title
- **Footer**: Page numbers, document version, date
- **Color Bar**: Thin accent line (2-4px) in Primary Dark Green, or Gold accent line

### Tables
- **Header Background**: Primary Dark Green (#1a4d2e)
- **Header Text**: White
- **Alternating Rows**: Light Gray (#f5f5f5)
- **Borders**: Light Gray (#e0e0e0)
- **Padding**: 8-12pt
- **Accent**: Gold (#d4af37) for highlights

### Checklists & Forms
- **Checkboxes**: Custom styled, Primary Dark Green when checked, Gold accent on hover
- **Input Fields**: Light border, rounded corners (4px)
- **Focus State**: Primary Dark Green border, 2px, or Gold accent
- **Labels**: Medium Gray, 11pt, Semi-Bold

## Document Templates

### Capability Statement
- **Format**: 8.5" x 11" or 8.5" x 5.5" (half sheet)
- **Header**: Logo, company name, UEI
- **Sections**: Color-coded with subtle backgrounds
- **Callouts**: Accent colors for differentiators

### Checklists
- **Format**: Vertical or 2-column
- **Visual Hierarchy**: Section headers, clear numbering
- **Checkboxes**: Large, easy to check
- **Spacing**: Generous white space

### Proposals
- **Format**: Standard letter size
- **Cover Page**: Full-color header, professional layout
- **Internal Pages**: Consistent headers/footers
- **Tables**: Professional formatting

### Presentations
- **Format**: 16:9 aspect ratio
- **Master Slides**: Consistent branding
- **Typography**: Large, readable fonts (24pt minimum)
- **Visuals**: High-quality images, minimal text per slide

## File Naming Conventions

### Naming Pattern
`[Document-Type]-[Topic]-[Version]-[Date].ext`

Examples:
- `checklist-sam-registration-v2-2024.pdf`
- `template-capability-statement-v1-2024.docx`
- `guide-apex-accelerator-v1-2024.pdf`

## Accessibility

### Requirements
- **Color Contrast**: WCAG AA compliant (4.5:1 for text)
- **Font Size**: Minimum 11pt for documents
- **Alt Text**: All images include descriptive alt text
- **Structure**: Proper heading hierarchy
- **PDFs**: Tagged for screen readers

## Implementation Guidelines

### Document Creation Tools
- **Microsoft Word**: For templates and documents
- **Google Docs**: For collaborative templates
- **Adobe InDesign**: For high-end design documents
- **Canva/Figma**: For graphics and infographics
- **PowerPoint**: For presentations

### File Formats
- **Primary**: PDF (universal compatibility)
- **Editable**: DOCX, XLSX, PPTX
- **Graphics**: PNG (transparency), JPG (photos)
- **Vector**: SVG (web), AI (design files)

## Brand Assets

### Logo Usage
- **Placement**: Top left or center header
- **Size**: Appropriate to document size
- **Clear Space**: 1x logo height around logo
- **Color**: Full color or Primary Blue only

### Watermarks
- **Usage**: Subtle background watermark for draft documents
- **Opacity**: 15-20%
- **Position**: Centered, slightly rotated

## Quality Standards

### Before Finalizing
- [ ] All colors match brand palette
- [ ] Typography is consistent
- [ ] Spacing is uniform
- [ ] All images are high resolution (300 DPI for print)
- [ ] All links work
- [ ] Document is properly formatted
- [ ] Spelling and grammar checked
- [ ] Accessibility standards met
- [ ] File name follows convention
- [ ] Version number updated
