# Brand Colors - Dark Green & Gold

## Primary Brand Colors

### Dark Green (Primary)
- **Hex**: `#1a4d2e`
- **RGB**: RGB(26, 77, 46)
- **Usage**: Primary brand color for headers, headings, main elements
- **Meaning**: Growth, Stability, Trust, Professionalism

### Gold (Accent)
- **Hex**: `#d4af37`
- **RGB**: RGB(212, 175, 55)
- **Usage**: Accent color for highlights, callouts, important elements
- **Meaning**: Excellence, Achievement, Premium Quality

## Secondary Colors

### Medium Green
- **Hex**: `#2d5016`
- **RGB**: RGB(45, 80, 22)
- **Usage**: Secondary elements, success states
- **Meaning**: Confidence, Success, Depth

### Light Green
- **Hex**: `#90c695`
- **RGB**: RGB(144, 198, 149)
- **Usage**: Backgrounds, subtle accents
- **Meaning**: Freshness, Growth, Positivity

## Neutral Colors

### Dark Gray (Text Primary)
- **Hex**: `#333333`
- **RGB**: RGB(51, 51, 51)
- **Usage**: Primary text color

### Medium Gray (Text Secondary)
- **Hex**: `#666666`
- **RGB**: RGB(102, 102, 102)
- **Usage**: Secondary text, captions

### Light Gray (Backgrounds)
- **Hex**: `#f5f5f5`
- **RGB**: RGB(245, 245, 245)
- **Usage**: Section backgrounds, cards

### White
- **Hex**: `#ffffff`
- **RGB**: RGB(255, 255, 255)
- **Usage**: Primary backgrounds, contrast

## Status Colors

### Success
- **Hex**: `#2d5016` (Medium Green)
- **Usage**: Success messages, positive indicators

### Warning
- **Hex**: `#d4af37` (Gold)
- **Usage**: Warnings, important notices

### Error
- **Hex**: `#c92a2a` (Red)
- **Usage**: Error messages, critical alerts

### Info
- **Hex**: `#1a4d2e` (Dark Green)
- **Usage**: Informational messages

## Color Combinations

### Primary Combination
- **Dark Green** + **Gold** = Professional, Trustworthy, Premium

### Alternative Combinations
- **Dark Green** + **Light Green** = Fresh, Growth-oriented
- **Gold** + **White** = Clean, Elegant, Premium
- **Dark Green** + **White** = Professional, Clean

## Usage Guidelines

### Headers & Titles
- Use Dark Green (#1a4d2e) for all major headings
- Use Gold (#d4af37) for accent elements in headers

### Body Text
- Use Dark Gray (#333333) for primary text
- Use Medium Gray (#666666) for secondary text

### Accents & Highlights
- Use Gold (#d4af37) sparingly for emphasis
- Use Light Green (#90c695) for subtle backgrounds

### Backgrounds
- White for primary backgrounds
- Light Gray (#f5f5f5) for section backgrounds
- Dark Green (#1a4d2e) for header sections

### Borders & Dividers
- Dark Green (#1a4d2e) for primary borders
- Gold (#d4af37) for accent borders
- Light Gray (#e0e0e0) for subtle dividers

## Accessibility

### Contrast Ratios (WCAG AA Compliant)
- Dark Green on White: 8.5:1 ✓
- Gold on White: 2.9:1 (use with caution, may need darker shade)
- Dark Gray on White: 12.6:1 ✓
- White on Dark Green: 8.5:1 ✓

### Recommendations
- Always use Dark Green for text on white backgrounds
- Use Gold for accents, not for large text blocks
- Ensure minimum 4.5:1 contrast for all text
- Test color combinations for accessibility
