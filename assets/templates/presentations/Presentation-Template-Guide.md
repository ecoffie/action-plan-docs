# PowerPoint Presentation Template Guide

## Design Specifications

### Slide Master Setup

#### Title Slide
- **Background**: Dark Green (#1a4d2e) gradient or solid
- **Title Text**: White, 44-54pt, Bold
- **Subtitle**: Gold (#d4af37), 28-32pt
- **Logo**: Top left or center, white version
- **Tagline**: Light gray/white, 18pt

#### Content Slides
- **Background**: White or very light gray (#f5f5f5)
- **Header/Footer Bar**: Dark Green (#1a4d2e), 60px height
- **Slide Title**: Dark Green (#1a4d2e), 32-36pt, Bold
- **Body Text**: Dark Gray (#333333), 18-24pt minimum
- **Accent Line**: Gold (#d4af37), 4px, under title

### Color Scheme

**Primary Colors:**
- Dark Green: #1a4d2e (Headers, primary elements)
- Gold: #d4af37 (Accents, highlights)

**Supporting Colors:**
- Medium Green: #2d5016
- Light Green: #90c695
- Dark Gray: #333333 (Text)
- Light Gray: #f5f5f5 (Backgrounds)
- White: #ffffff

### Typography

**Font Family**: 
- Primary: Calibri, Arial, or system sans-serif
- Headings: Bold, Dark Green
- Body: Regular, Dark Gray
- Minimum sizes: 18pt for body, 24pt for headings

### Design Elements

#### Headers/Footers
- **Header**: Dark Green bar, 60px height
- **Footer**: Optional - page number, logo, date
- **Logo Placement**: Top left in header bar (white logo)

#### Shapes & Graphics
- **Accent Shapes**: Gold (#d4af37)
- **Icons**: Dark Green or Gold, consistent style
- **Borders**: 2-4px, Dark Green or Gold
- **Shadows**: Subtle, professional

#### Charts & Graphs
- **Primary Color**: Dark Green (#1a4d2e)
- **Secondary Color**: Gold (#d4af37)
- **Accent Colors**: Medium Green, Light Green
- **Background**: White or Light Gray
- **Grid Lines**: Light Gray (#e0e0e0)

### Animation Guidelines

- **Keep it minimal** - Use sparingly
- **Professional transitions** - Fade or simple slide
- **Timing** - Fast (0.3-0.5 seconds)
- **Avoid** - Distracting animations, sounds, excessive motion

### Best Practices

1. **Consistency**
   - Use same fonts throughout
   - Maintain color scheme
   - Consistent spacing

2. **Readability**
   - Minimum 18pt font
   - High contrast text
   - Limit text per slide (6x6 rule: 6 bullets, 6 words each)

3. **Visual Hierarchy**
   - Clear title hierarchy
   - Use size and color for emphasis
   - White space is important

4. **Branding**
   - Logo on every slide (subtle)
   - Consistent color usage
   - Professional appearance

### PowerPoint Setup Instructions

1. **Create Slide Master**
   - View → Slide Master
   - Design header/footer with Dark Green bar
   - Set up title and content layouts

2. **Set Up Color Theme**
   - Design → Colors → Customize Colors
   - Set Dark Green (#1a4d2e) as Accent 1
   - Set Gold (#d4af37) as Accent 2

3. **Create Layouts**
   - Title slide layout
   - Section header layout
   - Content layouts (single, two-column)
   - Image/content layouts

4. **Save as Template**
   - File → Save As → PowerPoint Template (.potx)
   - Name: "Federal Contracting Roadmap Template.potx"
