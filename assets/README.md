# Assets Directory

This directory contains all supporting materials, resources, and assets for the Federal Contracting Roadmap Webinar Series.

## Directory Structure

### `/checklists/`
Checklists and action item lists for various phases and processes
- Phase completion checklists
- Registration checklists
- Compliance checklists

### `/guides/`
Detailed guides and handouts
- Step-by-step process guides
- Reference materials
- How-to documents

### `/templates/`
Templates and worksheets
- Document templates
- Worksheet templates
- Form templates

### `/marketing/`
Marketing and promotional materials
- Email templates
- Social media content
- Promotion guides

### `/resources/`
Resource directories and reference materials
- Links and references
- Tool directories
- Resource lists

### `/images/`
Images and graphics
- Logos and branding
- Infographics
- Presentation images
- Icons

### `/slides/`
Presentation slides
- PowerPoint files
- Slide decks
- Presentation templates

### `/videos/`
Video content
- Recorded webinars
- Tutorial videos
- Demo videos

### `/documents/`
PDFs and other documents
- Handouts
- Reference documents
- Downloadable materials

## Adding Your Assets

Place your existing assets in the appropriate subdirectories based on file type and purpose. Maintain consistent naming conventions for easy organization.

## Naming Conventions

Suggested naming format: `[category]-[description]-[date].ext`
Example: `guide-sam-registration-2024.pdf`
