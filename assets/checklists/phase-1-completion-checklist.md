# Phase 1: Setup Completion Checklist

Use this checklist to ensure you've completed all Phase 1 requirements before moving to Phase 2.

## Business Structure

### Business Structure Selection
- [ ] Determined business structure (Supplier, Service Provider, or Consultant)
- [ ] Business legally registered (LLC, Corporation, etc.)
- [ ] Business structure documented and understood
- [ ] Tax structure determined and appropriate

### DUNS and UEI
- [ ] Obtained DUNS number (if still needed)
- [ ] Understand UEI (Unique Entity Identifier) system
- [ ] UEI obtained through SAM.gov registration
- [ ] UEI documented and saved

### Professional Email
- [ ] Created professional business email domain
- [ ] Email addresses set up for key personnel
- [ ] Email signature created with business information
- [ ] Email system organized for federal contracting communications

## NAICS Code Identification

- [ ] Researched NAICS codes for your industry
- [ ] Identified primary NAICS code(s)
- [ ] Selected 2-5 NAICS codes that match your business
- [ ] Verified NAICS codes are correct for your services/products
- [ ] Documented NAICS code selections
- [ ] Ready to enter NAICS codes in SAM.gov

## SAM.gov Profile Creation

### Pre-Registration Preparation
- [ ] Gathered all required business information
- [ ] Have EIN/TIN ready
- [ ] Have bank account information ready
- [ ] Have business addresses ready
- [ ] Have contact information ready

### Registration Process
- [ ] Created SAM.gov user account
- [ ] Started entity registration
- [ ] Completed all required sections
- [ ] Entered NAICS codes correctly
- [ ] Uploaded required documents (if any)
- [ ] Submitted registration
- [ ] Received confirmation
- [ ] Saved UEI number
- [ ] Noted expiration date (1 year from registration)

### Post-Registration
- [ ] Verified registration is active (7-10 business days)
- [ ] Tested login access
- [ ] Reviewed all entered information for accuracy
- [ ] Set calendar reminder for renewal (30-60 days before expiration)

## DSBS Registration

- [ ] Understood what DSBS (Dynamic Small Business Search) is
- [ ] Registered business in DSBS
- [ ] Optimized DSBS profile
- [ ] Verified business appears in searches
- [ ] Updated DSBS profile with accurate information

## Apex Accelerator Engagement

### Initial Contact
- [ ] Found local Apex Accelerator
- [ ] Scheduled initial consultation
- [ ] Prepared for first meeting
- [ ] Brought required documentation

### Active Engagement
- [ ] Attended initial consultation
- [ ] Understood available services
- [ ] Registered for bid matching (if available)
- [ ] Scheduled follow-up appointments
- [ ] Identified counselor/point of contact

### Ongoing Relationship
- [ ] Know how to contact Apex Accelerator
- [ ] Understand how to request services
- [ ] Plan to attend training workshops
- [ ] Engaged with counselor regularly

## Capability Statement

### Development
- [ ] Created capability statement draft
- [ ] Included all essential sections:
  - [ ] Company information and contact details
  - [ ] Core competencies
  - [ ] Past performance (if available)
  - [ ] Differentiators
  - [ ] Certifications and compliance
  - [ ] NAICS codes
  - [ ] UEI number

### Review and Finalization
- [ ] Reviewed for accuracy and completeness
- [ ] Checked for spelling and grammar errors
- [ ] Ensured professional appearance
- [ ] Had others review for feedback
- [ ] Finalized capability statement
- [ ] Created both one-page and multi-page versions
- [ ] Saved in multiple formats (PDF, Word)
- [ ] Ready to use for meetings and submissions

## Phase 1 Documentation

### Documents Ready
- [ ] Business registration documents
- [ ] SAM.gov registration confirmation
- [ ] Capability statement finalized
- [ ] NAICS code documentation
- [ ] Apex Accelerator contact information
- [ ] All registration numbers saved securely

### Knowledge Check
- [ ] Understand federal contracting basics
- [ ] Know your business structure and type
- [ ] Understand NAICS code system
- [ ] Familiar with SAM.gov functionality
- [ ] Know how to access Apex Accelerator services
- [ ] Ready to use capability statement

## Ready for Phase 2

### Prerequisites Met
- [ ] SAM.gov registration active
- [ ] NAICS codes selected and entered
- [ ] Capability statement ready
- [ ] Apex Accelerator relationship established
- [ ] Professional email and communication systems ready
- [ ] Basic understanding of federal contracting process

### Mindset
- [ ] Understand Phase 1 is foundation for everything
- [ ] Ready to start finding opportunities (Phase 2)
- [ ] Committed to ongoing business development (Phase 3)
- [ ] Understand this is a journey, not a destination
